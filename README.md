# Dracmo KiCAD Component Library

This repository contains our KiCAD component library including:

- **Symbols**
- **Footprints**
- **3D Models**

The library is organized by **manufacturer**, and each symbol includes the following additional fields:

- `Part Number`
- `Manufacturer`
- `Supplier`
- `Supplier Part Number`

## Footprint Standards

All footprints are designed with the following conventions:

- **Silkscreen Text**:
  - Thickness: `0.15mm` with height `1mm`, or
  - Thickness: `0.13mm` with height `0.7mm`
- **Silkscreen Lines**:
  - Standard thickness: `0.2mm`
  - For extremely small components: `0.1mm`
- **Courtyards**:
  - Line thickness: `0.1mm`
- **3D Models**:
  - Included where available
- **Silkscreen and Courtyard**:
  - Properly drawn and aligned
